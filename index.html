üìù Commands You Actually Need to Know

üó∫Ô∏è Navigation (Moving Around)

Where The Hell Am I? - pwd

Show Current Location
pwd

üí° What's This Do?

pwd = "Print Working Directory" (fancy way of saying "where am I?")

It'll show something like: /data/data/com.termux/files/home

That's your home folder in Termux. Think of it as your personal space.

What's In Here? - ls

List All The Shit
ls -la

üí° The Options Breakdown

ls Show me what's here -l Long format (more details) -a ALL files (even hidden ones)

Pro tip: If nothing shows up, that's normal for a fresh install. Your folder is just empty, bro.

Moving Around - cd

This is how you navigate. Think of it like clicking folders, but with text:

cd ~ Go home
cd .. Go back
cd projects Enter folder

üíæ Accessing Your Phone Files

By default, Termux can't see your regular phone files. But we can fix that real quick:

‚ö†Ô∏è Heads Up

üîß When Shit Goes Wrong

Common Problems & How to Fix 'Em

üì± "Permission Denied" Bullshit

The Problem: Can't access some folders

The Fix: You probably forgot to run termux-setup-storage. Also, Termux can't access everything - work in your home folder or storage folders.

üì¶ "Package Not Found" Errors

The Problem: Can't install shit

The Fix:

  1. Run pkg update first
  2. Make sure you're connected to WiFi
  3. Try termux-change-repo to switch mirrors

‚å®Ô∏è Keyboard Acting Weird

The Problem: Can't use Ctrl or special keys

The Fix: Get Hacker's Keyboard from F-Droid, or use the Volume+Key combos I showed you earlier.
