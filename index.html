What is Termux?

Alright Forrest, let me break this down for you. Termux is basically Linux on your Android phone. No root needed, no complicated setup – just install and you've got a full Linux terminal in your pocket.

Why You'll Love This

Look, I know terminal stuff might seem intimidating, but here's what you'll be able to do:

  • Run Python scripts anywhere
  • SSH into servers from your phone
  • Automate boring tasks
  • Build and test code on the go
  • Look like a wizard to everyone else
$ python server.py * Running on http://127.0.0.1:8080 * Serving Flask app 'server' * Debug mode: on $ echo "Running Linux on my phone!" Running Linux on my phone!

First Launch Setup

What You'll See

When you open Termux for the first time, you'll see something like this:

Welcome to Termux! Wiki: https://wiki.termux.com Community forum: https://termux.com/community Gitter chat: https://gitter.im/termux/termux IRC channel: #termux on libera.chat Working with packages: * Search packages: pkg search <query> * Install a package: pkg install <package> * Upgrade packages: pkg upgrade Report issues at https://github.com/termux/termux-app/issues $

Quick Explanation

That dollar sign ($) is the prompt – it means Termux is ready for your commands. Think of it as the terminal saying "What's next?"

Essential First Command

Before we do anything else, let's update everything:

Update Termux
pkg update && pkg upgrade

What This Does

  • pkg update – Refreshes the list of available packages
  • && – Means "and then" (run the next command if successful)
  • pkg upgrade – Installs any available updates

When it asks "Do you want to continue? [Y/n]", just type Y and hit Enter.

Storage Setup

Forrest, this is important – by default, Termux can't see your phone's files. Let's fix that right now:

Enable storage access
termux-setup-storage

Permission Required

Android will ask for storage permission. Grant it to continue. This is safe – we're just connecting Termux to your files.

What This Creates

After running the command, you'll have these shortcuts in your home directory:

$ ls ~/storage/ dcim downloads movies music pictures shared
~/storage/downloads
Your Downloads folder
~/storage/dcim
Camera photos and videos
~/storage/shared
Main internal storage

Quick Example

Want to see your downloads? Just type:

List downloads
ls ~/storage/downloads

Terminal Basics

Forrest, think of the terminal as texting your phone. You type a command, hit Enter, and it responds or does something.

Essential Keyboard Shortcuts

Action Shortcut What It Does
Clear Screen Ctrl+L or clear Cleans up the terminal
Cancel Command Ctrl+C Stops whatever is running
Previous Command ↑ (up arrow) Recalls last command
Search History Ctrl+R Search through command history
Exit Termux exit or Ctrl+D Closes the terminal

Termux Touch Controls

Can't find the Ctrl key? Here's how to access special keys:

  • Swipe from left edge → Extra keys toolbar
  • Volume Up + Q → Ctrl+C
  • Volume Down + K → Up arrow
  • Or install Hacker's Keyboard from F-Droid

Installing Software

The package manager (pkg) is like an app store for Termux. Here's how to use it:

pkg search [name]
Find available packages
pkg install [name]
Install a package
pkg list-installed
See what's installed
pkg uninstall [name]
Remove a package

Essential Packages to Install

Here's my recommended starter pack:

Install all at once
pkg install git wget curl nano python nodejs openssh

What Each Package Does

  • git – Version control system
  • wget & curl – Download files from the internet
  • nano – User-friendly text editor
  • python – Python programming language
  • nodejs – JavaScript runtime
  • openssh – SSH client and server

Real Projects

Project 1: Python Script

Let's create a simple Python script:

Create and edit a Python file
nano hello_forrest.py

Add this code:

Python code
#!/usr/bin/env python3 import random import datetime name = "Forrest" now = datetime.datetime.now() print(f"Hey {name}! Welcome to Python on Termux!") print(f"Current time: {now.strftime('%Y-%m-%d %H:%M:%S')}") print(f"Your lucky number today is: {random.randint(1, 100)}")

Save (Ctrl+O), exit (Ctrl+X), then run it:

Run the script
python hello_forrest.py

Project 2: Web Server

Start a simple web server to share files:

One-line web server
python -m http.server 8080

How to Access Your Server

  1. Find your phone's IP: ip addr | grep wlan0
  2. On any device on the same network, open browser
  3. Go to: http://[your-phone-ip]:8080
  4. Press Ctrl+C to stop the server

Project 3: Organize Downloads

Here's a script to automatically organize your downloads:

organize_downloads.sh
#!/bin/bash # Forrest's Download Organizer cd ~/storage/downloads || exit echo "Organizing downloads..." # Create folders if they don't exist mkdir -p Images Documents Videos Music Archives # Move files to appropriate folders mv *.{jpg,jpeg,png,gif,webp} Images/ 2>/dev/null mv *.{pdf,doc,docx,txt} Documents/ 2>/dev/null mv *.{mp4,mkv,avi,mov} Videos/ 2>/dev/null mv *.{mp3,flac,m4a,wav} Music/ 2>/dev/null mv *.{zip,rar,7z,tar,gz} Archives/ 2>/dev/null echo "Downloads organized!" ls -la

Make it executable and run:

Make executable and run
chmod +x organize_downloads.sh && ./organize_downloads.sh

Advanced Topics

SSH Access

Control your phone from your computer via SSH:

Start SSH server
sshd

SSH Connection Info

  • Default port: 8022
  • Username: Get with whoami
  • Password: Set with passwd
  • Connect: ssh -p 8022 user@phone-ip

Troubleshooting

Permission Denied Errors

Problem: Can't access certain directories

Solution: Termux has limited permissions. Use termux-setup-storage and work within your home directory or storage folders.

Package Not Found

Problem: Can't install packages

Solutions:

  1. Update first: pkg update
  2. Check your internet connection
  3. Try changing repository: termux-change-repo

Keyboard Issues

Problem: Can't use special keys

Solutions:

  • Enable extra keys in Termux settings
  • Use Volume+Key combos
  • Install Hacker's Keyboard from F-Droid
